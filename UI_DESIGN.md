# 页面设计文档

## 1. 整体布局设计

### 1.1 管理后台布局

```
┌─────────────────────────────────────────────────────────┐
│  Logo    系统名称              用户信息  退出登录        │ 顶部导航栏 (60px)
├──────────┬──────────────────────────────────────────────┤
│          │                                              │
│  菜单    │                                              │
│  导航    │              内容区域                        │
│          │                                              │
│  - 仪表盘│                                              │
│  - 发现页│                                              │
│  - 频道  │                                              │
│  - 系统  │                                              │
│          │                                              │
│          │                                              │
└──────────┴──────────────────────────────────────────────┘
  200px                    自适应宽度
```

**布局参数**:
- 顶部导航栏高度: 60px
- 左侧菜单宽度: 200px (可折叠)
- 内容区域: 自适应
- 响应式: 移动端菜单折叠

## 2. 页面设计

### 2.1 列表页面设计

**通用结构**:
```
┌─────────────────────────────────────────────────────────┐
│  页面标题                         操作按钮区            │
├─────────────────────────────────────────────────────────┤
│  筛选条件: 频道选择 | 搜索框 | 类型筛选 | 状态筛选    │
├─────────────────────────────────────────────────────────┤
│  数据表格                                              │
│  ┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐   │
│  │ 序号│ ID  │ 名称│ 类型│ 状态│ 时间│ 描述│ 操作│   │
│  ├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤   │
│  │  1  │ 22  │ xxx │ xxx │ 启用│ xxx │ xxx │ 编辑│   │
│  │  2  │ 23  │ xxx │ xxx │ 禁用│ xxx │ xxx │ 删除│   │
│  └─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘   │
├─────────────────────────────────────────────────────────┤
│  分页器: 上一页 1 2 3 4 5 下一页    每页10条 共100条   │
└─────────────────────────────────────────────────────────┘
```

**组件选择**:
- 表格: `a-table` (Ant Design Vue)
- 分页: `a-pagination`
- 筛选: `a-select`, `a-input-search`
- 按钮: `a-button`
- 标签: `a-tag`

**交互设计**:
- 点击表头排序
- 点击行选中
- 双击行编辑
- 右键菜单
- 批量操作

## 6. 响应式设计
