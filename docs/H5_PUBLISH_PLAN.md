# H5发布功能与话题标签管理开发计划

## 概述

本计划包含两个主要模块的开发：
1. **话题与标签管理** - Admin后台管理功能
2. **H5发布功能** - 支持6种物料类型的发布

## 五、开发优先级

1. **话题/标签后端API** - 提供数据支持
2. **话题/标签Admin前端** - 运营可管理
3. **H5发布页面框架** - 基础架构
4. **图文发布** - 最简单的类型，验证流程
5. **视频发布** - 核心功能
6. **短剧/漫剧/小说发布** - 复杂类型
7. **测试数据生成** - 验证展示效果
8. **全面测试** - 确保质量

## 六、技术要点

### 6.1 文件上传

- 使用 FormData 进行文件上传
- 支持大文件分片上传（视频）
- 上传进度显示
- 图片压缩处理

### 6.2 发布流程

```
用户操作 → 表单验证 → 文件上传 → 提交API → 发布成功 → 跳转详情
```

### 6.3 数据结构

```typescript
// 发布请求
interface PublishRequest {
  type: 'image_text' | 'long_video' | 'short_video' | 'short_drama' | 'drama' | 'novel';
  title: string;
  subtitle?: string;
  description?: string;
  cover_url?: string;
  content_url?: string;
  category?: string;
  tags?: string[];
  topic_ids?: number[];
  // 各类型特有字段...
}
```
