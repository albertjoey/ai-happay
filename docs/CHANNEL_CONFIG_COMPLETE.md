## 🎯 配置详情

### 金刚位配置 (每个频道5个)
```json
{
  "title": "热门话题",
  "icon": "🔥",
  "link_type": "topic",
  "link_value": "1",
  "sort": 1
}
```

**链接类型**:
- topic: 话题
- content: 内容
- rank: 排行榜
- activity: 活动
- more: 更多

### 推荐位配置 (每个频道3个)
```json
{
  "title": "今日推荐",
  "display_type": "single",
  "source_type": "algorithm",
  "content_ids": [1, 2, 3, 4, 5, 6],
  "materials": [...] // 包含完整物料信息
}
```

**展示类型**:
- single: 单图大图
- scroll: 横向滑动
- grid: 网格布局

**内容来源**:
- algorithm: 算法推荐
- manual: 人工推荐
- filter: 条件筛选

### Feed流配置 (每个频道1个)
```json
{
  "title": "推荐Feed",
  "layout_type": "two_col",
  "content_strategy": "algorithm",
  "content_ids": [1, 2, 3, 4, 5, 6],
  "materials": [...] // 包含完整物料信息
}
```

**布局类型**:
- two_col: 双列瀑布流

**内容策略**:
- algorithm: 算法推荐

### 广告位配置 (每个频道2个)
```json
{
  "name": "频道固定广告",
  "insert_type": "fixed",
  "insert_rule": {
    "fixed_position": 3,
    "max_count": 1
  },
  "ad_type": "image",
  "ad_content": {
    "image_url": "https://picsum.photos/400/600?ad=2-1",
    "title": "精彩广告"
  }
}
```

**插入方式**:
- fixed: 固定位置(第3条后插入)
- interval: 间隔插入(每5条插入一个)

**广告类型**:
- image: 图片广告
