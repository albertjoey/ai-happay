## 🎯 广告插入功能详解

### 实现方式

#### 1. 固定位置插入
```typescript
if (ad.insert_type === 'fixed') {
  const position = ad.insert_rule.fixed_position || 3;
  if (index + 1 === position) {
    // 在第3条内容后插入广告
  }
}
```

#### 2. 间隔插入
```typescript
if (ad.insert_type === 'interval') {
  const interval = ad.insert_rule.interval || 5;
  if ((index + 1) % interval === 0) {
    // 每5条内容插入一个广告
  }
}
```

#### 3. 随机插入
```typescript
if (ad.insert_type === 'random') {
  const randomInterval = Math.floor(Math.random() * 3) + 3;
  if ((index + 1) % randomInterval === 0) {
    // 随机间隔插入广告
  }
}
```
